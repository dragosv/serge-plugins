=encoding utf8

=head1 NAME

Serge::Sync::Plugin::TranslationService::delta - L<Delta|https://github.com/dragosv/delta> synchronization plugin for L<Serge|https://serge.io/> based on L<Delta CLI|https://github.com/dragosv/delta>

=head1 SYNOPSIS

    ts
    {
        plugin                      crowdin

        data
        {
            config_file              ./files/config/delta_config_file.yml
        }
    }

=head1 DESCRIPTION

Integration between Serge (Free, Open Source Solution for Continuous Localization) and Delta (Pluggable Command Line Interface for job-based Translation Management Services),
implemented using the command-line client for the Delta.

=head1 ATTRIBUTES

=over

=item I<config_file>

Delta CLI path to the configuration file

=back

=head1 INSTALLATION

 cpanm Serge::Sync::Plugin::TranslationService::delta
